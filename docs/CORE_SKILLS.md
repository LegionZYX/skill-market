## 📈 优先级说明

- **P0 (核心必备)**: 20个技能 - 企业/个人高频使用,优先开发
- **P1 (常用推荐)**: 40个技能 - 常见场景,次优先开发
- **P2 (可选)**: 暂不收录 - 特定场景,按需开发

## 🎯 质量标准

1. **评分 ≥ 4.2** - 所有技能平均评分在4.2以上
2. **真实场景** - 每个技能都有明确的业务价值
3. **高频使用** - 优先保留下载量高的技能
4. **平台覆盖** - 支持主流平台,避免单一依赖
