## 🚀 使用方法

### 1. 查看网页版（最快）

```bash
# 方法1: 直接打开
cd skill-marketplace/web
# 双击 index.html

# 方法2: 本地服务器
python -m http.server 8000
# 访问 http://localhost:8000
```

### 2. 查看完整数据

```bash
# JSON格式（推荐）
cat data/skills_v4_index.json

# CSV格式（Excel）
打开 data/skills.csv

# Markdown文档
查看 docs/SKILL_CATALOG_V4.md
```

### 3. 部署到线上

```bash
# GitHub Pages（免费）
git push origin main
# Settings → Pages → Enable

# Vercel（推荐）
cd web && vercel

# Netlify（拖拽）
拖拽 web/ 文件夹到 netlify.com
```
